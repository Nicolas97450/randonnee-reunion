# Pages légales — Randonnée Réunion

Ce dossier contient les pages légales hébergées via GitHub Pages.

## Setup

1. Aller dans les Settings du repo GitHub → Pages
2. Source : Deploy from a branch
3. Branch : `main`, dossier `/legal-pages`
4. Sauvegarder
5. Configurer le domaine custom `randonnee-reunion.re` (nécessite un CNAME DNS)

## URLs finales

- `https://randonnee-reunion.re/confidentialite` → Politique de confidentialité
- `https://randonnee-reunion.re/cgu` → Conditions générales d'utilisation

## Fichiers

- `index.html` — Page d'accueil avec liens vers les documents
- `confidentialite.html` — Politique de confidentialité (RGPD)
- `cgu.html` — Conditions générales d'utilisation
- `CNAME` — Configuration du domaine custom (à modifier si domaine différent)
