HIGH (a corriger avant la beta)

H1 — 3 violations TypeScript strict (as any)
ReportForm.tsx:120, FeedScreen.tsx:389, NavigationCTA.tsx:10
CODE La regle du projet interdit any. Ces 3 cast as any contournent le typage. Les 2 premiers sont lies a FormData (upload fichiers), le 3eme a un type d'animation.
Action : Creer une interface FormDataFile proprement typee. Remplacer Record<string, any> par Record<string, number>. Temps : 2h.

SECURITE Dependances transitives via Expo/React Native. Les 2 HIGH concernent xmldom (injection XML) et node-forge (crypto). Ce sont des deps dev mais a surveiller.
Action : Tester npm audit fix + verifier la compatibilite. Si non fixable, documenter dans TASKS.md. Temps : 1h.
H3 — 2 console.error non proteges par __DEV__
useTrailReports.ts:71, useTrailReviews.ts:106
CODE Ces console.error sont visibles en production. Ils devraient etre proteges par __DEV__ ou reportes a Sentry.
Action : Ajouter __DEV__ && devant ou remplacer par Sentry.captureException(). Temps : 30 min.
