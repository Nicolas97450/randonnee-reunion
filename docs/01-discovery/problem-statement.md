# Problem Statement — Randonnée Réunion

## Le problème

La Réunion possède un réseau exceptionnel de 710+ sentiers de randonnée, mais il n'existe aucune application mobile moderne et complète dédiée à l'île. Les randonneurs réunionnais et les touristes sont obligés de jongler entre plusieurs outils :

- **Randopitons** : référence locale mais UX vieillissante, pas d'app mobile native
- **AllTrails** : app moderne mais seulement 495 sentiers Réunion, pas de données locales (ONF, météo montagne)
- **Visorando** : seulement 17 sentiers Réunion
- **Groupes WhatsApp** : pour organiser les sorties (chaotique, pas adapté)

## La cible

- Randonneurs réunionnais (population : 900K habitants, ~30% pratiquent la rando)
- Touristes (500K+ visiteurs/an, rando = activité n°1)
- Clubs et associations de randonnée locaux

## La proposition de valeur

**La première app qui te donne envie de découvrir toute La Réunion — pas juste de t'y retrouver.**

- 710 sentiers avec données terrain exactes (vs 17-495 chez les concurrents)
- Météo montagne enrichie (UV, rafales, visibilité) + statut ONF en temps réel
- Gamification territoriale : "colorier" l'île en complétant les sentiers
- Sorties de groupe avec chat temps réel
- GPS fiable en montagne avec guidage vocal
