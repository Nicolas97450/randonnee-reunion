# User Personas — Randonnée Réunion

## Persona 1 : Marie, la randonneuse régulière

- **Âge** : 32 ans
- **Lieu** : Saint-Denis, La Réunion
- **Profil** : Enseignante, randonne 2-3 fois/mois
- **Besoins** : Trouver de nouveaux sentiers, connaître l'état ONF, météo fiable
- **Frustrations** : Randopitons est vieillot, AllTrails manque de sentiers locaux
- **Usage** : Cherche des sentiers par région/difficulté, vérifie la météo avant de partir
- **Objectif** : Découvrir tous les sentiers de l'île progressivement

## Persona 2 : Thomas, le randonneur occasionnel / touriste

- **Âge** : 28 ans
- **Lieu** : Métropole, visite La Réunion 2 semaines
- **Profil** : Développeur web, sportif, découvre La Réunion
- **Besoins** : Sentiers populaires, GPS fiable en montagne, itinéraire vers le départ
- **Frustrations** : Pas de réseau en montagne, se perd facilement
- **Usage** : Suit les sentiers recommandés, utilise le GPS tracking
- **Objectif** : Faire les incontournables sans se perdre

## Persona 3 : Éric, le randonneur social

- **Âge** : 45 ans
- **Lieu** : Saint-Pierre, La Réunion
- **Profil** : Cadre, organise des randos en groupe le week-end
- **Besoins** : Organiser des sorties de groupe, chat temps réel, partager des photos
- **Frustrations** : Coordonner via WhatsApp est chaotique
- **Usage** : Crée des sorties, invite ses amis, utilise le chat pendant la rando
- **Objectif** : Fédérer une communauté locale de randonneurs
